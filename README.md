## Тема

## План работ
1. Оформление требований по работе
2. Разработка серверной части
3. Разработка клиентской части
4. Тестирование системы на эмуляции (по возможности с датчиками)

## Требования (общие)
### Клиент
1. Мониторинг состояния сервера (дашбоард)
2. Просмотр истории состояния
3. Выгрузка данных с сервера
4. Изменение режима управления: ручное/автоматическое

### Сервер (Модель)
1. Сбор данных с датчиков: (для примера - температура)
2. Управление программным реле
3. Хранение истории датчиков
4. эмуляция работы реле (подача сигнала на лампу)

## Требования (технические)
Сервер: OrangePi Zero2W + Flask+SQLAlchemy+sqlite+http
Клиент: Android + Java+http + ...
